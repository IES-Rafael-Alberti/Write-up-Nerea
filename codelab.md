# Write-up de Mis Máquinas

## Sección 1: Introducción
Duration: 0:05:00

Bienvenido a este Codelab donde documentaremos el análisis y resolución de mis máquinas de práctica para el examen.
Aquí puedes ver como he resolvido varias máquinas de windows

### Cajas de información
Positive
: Este Codelab te ayudará a llevar un registro ordenado de tus avances y estrategias.

Negative
: No seguir los pasos puede provocar que pierdas información importante de tus write-ups.

### Lista con viñetas
* Conectar a la máquina
* Enumerar servicios
* Documentar hallazgos
* Resolver la flag

### Lista numerada
1. Conectar a la máquina vía SSH/VPN
2. Realizar reconocimiento de servicios
3. Explorar y obtener información de usuario y root
4. Guardar todas las notas en este Codelab
